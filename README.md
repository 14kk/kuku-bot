## 功能
* QQ自动签到
* 超级萌宠快速养成
* 自动修改步数
* 网易云音乐打卡和每日300首听歌量
* 一些小工具

指令：[https://u.iheit.com/kuku/bot/menu.html](https://u.iheit.com/kuku/bot/menu.html)

安装教程：[https://blog.kuku.me/index.php/archives/3/](https://blog.kuku.me/index.php/archives/3/)
